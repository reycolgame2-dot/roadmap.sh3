#Personal Portofolio

A Responsive personal portofolio website built with HTML and CSS as part of the roadmap.sh Frontend Projects

## Features

- Responsive layout for desktop and mobile
- Semantic HTML5 structure
- CSS flexbox and Grid
- Contact page
- Multi-page navigation

## Technologies used

- HTML5
- CSS3
- Google Fonts (Poppins)

## Project Structure


```
.
├── assets/
├── css/
│   ├── style.css
│   └── responsive.css
├── index.html
├── projects.html
├── articles.html
├── contact.html
├── favicon.ico
└── README.md
```

## How to run

1. Clone this repository

```bash
git clone https://github.com/ReyhanAllofadieka/personal-portfolio.git
```

2. Open the project folder.

3. Open `index.html` in your browser or use Live Server in VS Code.
